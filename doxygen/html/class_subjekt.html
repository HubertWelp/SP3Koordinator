Detailed Description

Klasse zur Verwaltung von Beobachtern im Observer-Pattern.

Klassen sollten von Subjekt abgeleitet werden, falls sie in Verbindung mit Beobachter-Klassen das Observer-Pattern implementieren wollen. Subjekt stellt die Funktionalitaet zur Verwaltung der Beobachter zur Verfuegung.

benachrichtigt alle Beobachter ueber Aenderungen
