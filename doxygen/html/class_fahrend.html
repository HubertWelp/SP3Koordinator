Detailed Description

Die Klasse bildet den "Fahrend"-Zustand im Patientendialogsystem ab.

Die Dialogfuehrung im Patientendialogsystem wird ueber ein Zustandsmuster realisiert. Die Klasse Fahrend stellt die Funktionalitaet und die Daten fuer den Zustand Fahrend zur Verfuegung. Dies umfasst die Implementierung der Ereignisse "angekommen" und "fahrenAbgebrochen". Dieser Zustand wird angenommen, waehrend der Roboter einen Patienten zu einem ausgewaehlten Ziel geleitet.
